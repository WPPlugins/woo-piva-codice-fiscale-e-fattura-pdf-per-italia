Il plugin che ti permette di rendere WooCommerce adatto al mercato italiano con l’inserimento del C.F. e della Partita IVA

== Description ==

WC requires at least: 2.2<br />
WC tested up to: 2.3<br /><br />
Il plugin <strong>WooCommerce P.Iva e Codice Fiscale per Italia</strong> permette di rendere <i>WooCommerce</i> adatto al mercato italiano grazie all’inserimento del codice fiscale e della partita IVA del cliente.<br /><br />
Grazie al plugin è possibile:
<ul>
<li>Inserire in fase di checkout i campi Codice Fiscale e Partita IVA</li>
<li>Scegliere la tipologia di cliente (Privato, Azienda, Libero Professionista) che sta effettuando l’acquisto</li>
<li>Modificare in amministrazione i campi Codice Fiscale e Partita IVA</li>
<li>Ricevere nel dettaglio ordine e via mail i campi C.F e P.IVA creati</li>
<li>Generare automaticamente la fattura/ricevuta lato amministrazione grazie al plugin <i>WooCommerce PDF Invoices & Packing Slips</i>.</li>
</ul>


== Installation ==

1. Caricare la cartelle del plugin all'interno di `/wp-content/plugins/` oppure installare il plugin attraverso la sezione WordPress plugins.
1. Attivare il plugin attraverso la sezione 'Plugins' di WordPress.
